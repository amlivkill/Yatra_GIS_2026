## 📝 Description
**GIS AGENT** ek advanced mapping aur spatial analysis tool hai jo **Millet Mission 2026** aur **Kedarnath Dham Yatra 2026** ke safal sanchalan ke liye banaya gaya hai. Yah project Uttarakhand ke **Rudraprayag aur Tehri** blocks mein rural livelihood enhancement, climate-resilient agriculture aur administrative monitoring ko AI automation ke jariye badhava deta hai.

Yah tool vishesh roop se 14 yatra sectors ki sanitation, medical facilities, aur logistics mapping ke liye District Administration Rudraprayag ke sahyog se design kiya gaya hai.

## 🚀 Key Features
* **AI Engine:** Gemini Flash 2.0 / GPT-4o Integration — jo data se real-time insights aur reporting generate karta hai.
* **Sector Mapping:** 14 Yatra sectors (Sirobagarh se Kedarnath Mandir tak) ki department-wise (Police, Health, Water, Cleaning) live tracking.
* **Location Focus:** Uttarakhand (Rudraprayag & Tehri Blocks) — Himalayan topography ke anusaar customized.
* **Master Dashboard:** "Changetach Studio" ke sath seamless connectivity.
* **Carbon Credit Advisor:** Sustainable agriculture aur forest management ke liye integrated advisor bot.

## 🛠️ Tech Stack
* **Language:** Python 3.13
* **Backend:** Flask (Python based API)
* **Deployment/Tunneling:** Ngrok (For secure local-to-cloud access)
* **Frontend:** Leaflet.js (GIS Mapping), HTML5, CSS3, JavaScript
* **Data Format:** JSON & CSV (Compatible with Excel/GIS compiled data)

## 📁 Project Structure
- `/js`: GIS layers (districts.js, microgrid.js, yatra.js)
- `/css`: Premium dark-themed UI styling
- `index.html`: Master Hub Dashboard
- `yatra_rudraprayag.html`: Focused mapping for 14 sectors
